# Deep-learning-vgg16-cnn
Εκπαίδευση μοντέλων μηχανικής μάθησης (vgg16, cnn) με σκοπό την ταξινόμηση ακτινογραφιών θώρακος σε τρεις κατηγορίες: (α) υγιής, (β) πνευμονία λόγω βακτηριδιακής λοίμωξης (γ) πνευμονία λόγω ιού. Για την εργασία χρησιμοποίηθηκε δείγμα 4600 εικόνων με ακτινογραφίες εκ των οποίων οι 1100 ήταν οι τεστ εικόνες και 3500 οι train εικόνες. Τα δύο μοντέλα vgg16 και cnn έβγαλαν ποσοστό επιτυχίας 82 και 75 τις εκατό αντίστοιχα στο δείγμα τεστ που δόθηκε στον διαγωνισμό kaggle στον οποίο έτρεξε. Για την εργασία χρησιμοποιήθηκε η γλώσσα προγραμματισμού Python στο περιβάλλον colab της google όπου δίνεται ένα virtual machine με εξαιρετικά χαρακτηριστικά ώστε να τρέξουν βαριά μοντέλα όπως τα δύο παραπάνω σε μικρό χρονικό διάστημα. Επίσης χρησιμοποιήθηκαν οι βιβλιοθήκες tensorflow-keras.
